# e2e/

Účel: end-to-end testy (napr. Playwright) na overenie celých user flow.

Čo sem pôjde:
- Scenáre pre hlavné cesty (Home → Kontakt, Privacy/Cookies odkazy, projekty)
- Kontrola dostupnosti a základnej accessibility (napr. ARIA role)
- Prípadne screenshot/regresné testy, ak bude nasadené vizuálne testovanie

Poznámky:
- Momentálne prázdne; vytvorte Playwright config a scenáre, keď pribudne nasadenie alebo backend.
